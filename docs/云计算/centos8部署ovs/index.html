Centos8部署openvswitch

2023-11-20
1分钟阅读时长

安装依赖

yum groupinstall -y "Development Tools"
yum install -y dnf-plugins-core epel-release
yum config-manager --set-enabled powertools
yum install -y python3-sphinx python3-six numactl-devel libbpf-devel  libbpf libxdp-devel libxdp wget openssl-devel gcc make python3-devel openssl-devel kernel-devel graphviz kernel-debug-devel autoconf automake rpm-build redhat-rpm-config libtool deskto NetworkManager

构建rpm包

mkdir  -p  ~/rpmbuild/SOURCES
cd  /root/rpmbuild/SOURCES/
wget https://www.openvswitch.org/releases/openvswitch-3.2.1.tar.gz
tar xzf openvswitch-3.2.1.tar.gz
rpmbuild -bb --nocheck openvswitch-3.2.1/rhel/openvswitch-fedora.spec

安装

cd /root/rpmbuild/RPMS/x86_64/
yum localinstall -y openvswitch-3.2.1-1.el8.x86_64.rpm

配置ovs网桥

创建名为ovs_br0的网桥

ovs-vsctl add-br ovs_br0

将物理网卡ens18添加到 ovs_br0

ovs-vsctl add-port ovs_br0 ens18
ovs-vsctl show

使用os-net-config配置ovs网络

每次输命令配置ovs网络太麻烦了，所以我们使用openstack官方推出的os-net-config工具配置ovs网络，这是以yaml配置文件形式定义的

安装os-net-config

pip3 install os-net-config

安装的时候提示PyYAML版本不对，那强制安装下PyYAML这个包再安装os-net-config

pip3 install PyYAML --ignore-installed

创建配置文件目录

mkdir /etc/os-net-config

在这下面创建config.yaml,内容如下：

network_config:
  - type: ovs_bridge
    name: ovs-br0
    use_dhcp: false
    use_dhcpv6: false
    addresses:
    - ip_netmask: 10.249.2.102/24
    routes:
      - ip_netmask: 10.249.2.0/24
        next_hop: 10.249.2.1
        default: true
    dns_servers: 223.5.5.5
    members:
      -
        type: interface
        name: ens18

这是将绑定单张物理网卡到ovs-br0网桥的配置文件，如果想要其他的像VLAN之类的定义，请参考 链接

执行命令 os-net-config即可生效

但发现开机并不会自动启动，需要重启network服务,新建 /usr/local/bin/restart_network.sh 并赋予可执行权限：

#!/bin/bash
sleep 5
systemctl restart network

打开 /etc/rc.local 文件，并在文件的末尾添加以下行：

/usr/local/bin/restart_network.sh &

通过查看network-scripts里的脚本可知，这个工具修改了network-scripts的network配置文件

虚拟机相关配置

新建 /etc/libvirt/qemu/networks/ovs.xml:

<network>
  <name>ovs</name>
  <forward mode="bridge"/>
  <bridge name="ovs-br0"/>
  <virtualport type="openvswitch"/>
</network>

将配置文件加到virsh网络中

virsh net-define /etc/libvirt/qemu/networks/ovs.xml

查看

virsh net-list --all
