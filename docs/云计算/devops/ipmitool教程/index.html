Ipmitool 教程

2023-11-07
1分钟阅读时长

用户配置

查看用户列表

ipmitool user list 1

可以看到有个管理员ID 为2 的用户名为zbwdzsjq , 需要注意的是不一定所有服务器的管理员ID为 1或者2 ，所以需要根据上面的命令实际查看

创建用户

格式： ipmitool user set name 用户ID  用户名

设置密码

格式： ipmitool user set password  用户ID号  密码

给用户权限

格式：ipmitool channel setaccess 1 用户ID callin=on ipmi=on link=on privilege=值
privilege的值：
1 callback 
2 user 
3 operator 
4 administrator 
5 OEM

启用/禁用用户

格式： ipmitool user enable/disable 用户ID

查看授权

格式：ipmitool channel getaccess 1 用户ID

远程访问

格式：ipmitool -I lanplus -H ip地址 -U 用户名 -P 用户密码 命令

系统信息

查看功耗信息

ipmitool dcmi power reading

查看电源状态

ipmitool chassis power status

开机

ipmitool chassis power on

关机

ipmitool chassis power off

重启

ipmitool chassis power reset

查看SDR信息

ipmitool sdr

查看传感器信息

ipmitool sensor list

网络

查看网络

ipmitool lan print 1

修改网络

设置BMC IPV4地址为 10.1.10.201 掩码255.255.255.0 网关10.1.10.1

ipmitool lan set 1 ipaddr 10.1.10.201  
ipmitool lan set 1 netmask 255.255.255.0 
ipmitool lan set 1 defgw ipaddr 10.1.10.1
