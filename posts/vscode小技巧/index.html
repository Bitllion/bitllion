1. 用户代码片段

利用快捷键快速添加一段代码片段

vscode左下角-用户代码片段-选择python，打开一个json文件，替换成如下：

{
	"HEADER": {
        "prefix": "header",
        "body": [
            "#!/opt/yeepack/venv/bin/python3",
            "# -*- encoding: utf-8 -*-",
            "'''",
            "@文件        :$TM_FILENAME",
            "@说明        :",
            "@时间        :$CURRENT_YEAR/$CURRENT_MONTH/$CURRENT_DATE $CURRENT_HOUR:$CURRENT_MINUTE:$CURRENT_SECOND",
            "@作者        :fanyq",
            "@版本        :1.0",
            "'''",
            "import pretty_errors",
            "",
            "$0"
        ],
    }
}

在新的文件中键入“HEADER“或者”header ”，使用tabline自动联想或者tab联想，图标为方形的既是我们要的代码片段

2. python虚拟环境

shift+ctrl+p 输入workspack,选择打开工作区设置，键入以下json信息：

{
    "code-runner.fileDirectoryAsCwd": true,
    "python.defaultInterpreterPath": "/opt/yeepack/venv/bin/python3",
}

3. Coder Runner

修改默认python解释器

扩展-Coder Runner -扩展设置，点击任一在settings.json编辑，修改如下参数：

"code-runner.executorMap": {
    "python": "/opt/yeepack/venv/bin/python3 -u"
},
"python.envFile": "/opt/yeepack/backend/yeepackEnv",
"code-runner.defaultLanguage": "/opt/yeepack/venv/bin/python3"

以当前的文件所在目录执行
